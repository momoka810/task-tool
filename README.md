# タスク優先度ツール

人間的な判断軸を使ってタスクの優先順位を自動で整理するツールです。AIがあなたの将来像と現在の状況を考慮して、各タスクを5つの評価軸で評価し、最適な優先順位を提案します。

## ✨ 機能

- **AI自動評価**: タスク名と説明を入力するだけで、AIが自動で各評価軸を1〜5点で評価
- **将来像・現状考慮**: あなたの将来像と現在の状況を考慮した評価
- **重み付きスコア計算**: 汎用性と習得コストを重視した優先順位付け
- **理由の自動生成**: なぜその順位になったかの理由を自動で生成

## 🎯 特徴

- **簡単操作**: タスク名と説明だけを記入すればOK
- **人間的な判断**: 5つの評価軸（将来性、ギャップ、習得コスト、汎用性、心理的ハードル）で評価
- **安全なAPIキー管理**: `.gitignore`でAPIキーを保護
- **柔軟な設定**: 将来像と現在の状況を自由に記入可能

## 🔧 必要な環境

- Python 3.7以上
- OpenAI APIキー（[取得方法](https://platform.openai.com/api-keys)）
- openaiライブラリ

## 📦 セットアップ

### 1. リポジトリのクローン

```bash
git clone https://github.com/momoka810/task-tool.git
cd task-tool
```

### 2. 必要なライブラリのインストール

```bash
pip install openai
```

### 3. APIキーの設定

`api_key.txt`ファイルを作成し、OpenAI APIキーを記入してください：

```bash
nano api_key.txt
```

**記入方法：**
- APIキーのみを記入: `sk-proj-xxxxxxxxxxxxx`
- または環境変数形式: `OPENAI_API_KEY=sk-proj-xxxxxxxxxxxxx`（どちらでもOK）

**注意**: `api_key.txt`は`.gitignore`で除外されているため、Gitにコミットされません。

## 🚀 使い方

### 1. 将来像と現在の状況を記入

`5-2_オリジナル タスク優先度ツール.py`ファイルを開き、以下のセクションを編集してください：

```python
将来像 = """
あなたが目指している将来の姿や目標を記入してください
"""

現在の状況 = """
あなたの現在のスキルレベルや状況を記入してください
"""
```

### 2. タスクリストを編集

`tasks`リストに、やりたいタスクを追加してください：

```python
tasks = [
    {
        'タスク名': 'タスクの名前',
        '説明': 'タスクの詳細な説明'
    },
    # 新しいタスクを追加...
]
```

### 3. 実行

```bash
python "5-2_オリジナル タスク優先度ツール.py"
```

### 4. 結果の確認

実行すると、以下のような結果が表示されます：

- タスク優先順位ランキング（総合スコア順）
- 各タスクの優先順位の理由
- 各タスクの詳細評価（各評価軸のスコア）

## 📊 評価軸について

AIは以下の5つの評価軸で各タスクを1〜5点で評価します：

1. **将来性**（高いほど良い）
   - このタスクが将来にどれだけ役立つか

2. **今の自分とのギャップ**（小さいほど良い）
   - 現在のスキルとの差。小さいほどすぐに取り組める

3. **習得コスト**（低いほど良い）
   - 学習に必要な時間・労力。低いほど短時間で習得できる

4. **汎用性**（高いほど良い）
   - 他の場面でも使えるか。高いほど多くの場面で活用できる

5. **心理的ハードル**（低いほど良い）
   - 取り組む際の心理的な難しさ。低いほど始めやすい

## ⚠️ 注意事項

- **APIキーの管理**: `api_key.txt`ファイルは絶対にGitにコミットしないでください（`.gitignore`で除外済み）
- **API使用料**: OpenAI APIの使用には料金がかかります（gpt-4o-miniは比較的安価）
- **将来像・現状の記入**: より詳しく記入すると、AIの評価精度が向上します
- **インターネット接続**: AI評価にはインターネット接続が必要です
