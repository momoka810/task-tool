# タスク優先度ツール

将来像と現在の状況をもとに、AIがタスクの優先順位を自動で整理するツール。
5つの評価軸でスコアリングし、「今やるべきこと」を提案します。

## 機能

- 将来像・現在の状況を入力するとAIが各タスクを評価
- 5つの評価軸（将来性・ギャップ・習得コスト・汎用性・心理的ハードル）で1〜5点採点
- 重み付きスコアで優先順位を自動ランキング
- なぜその順位になったかの理由も出力

## 評価軸

| 評価軸 | 説明 |
|--------|------|
| 将来性 | 将来にどれだけ役立つか |
| 今とのギャップ | 現在のスキルとの差（小さいほど◎） |
| 習得コスト | 必要な時間・労力（低いほど◎） |
| 汎用性 | 他の場面でも使えるか |
| 心理的ハードル | 取り組む際の難しさ（低いほど◎） |

## セットアップ

```bash
git clone https://github.com/momoka810/task-tool.git
cd task-tool
pip install openai
```

`api_key.txt` を作成してOpenAI APIキーを記入：

```
sk-proj-xxxxxxxxxxxxx
```

## 使い方

`main.py` を開いて将来像・現在の状況・タスクリストを編集：

```python
将来像 = """
目指している将来の姿や目標を記入
"""

現在の状況 = """
現在のスキルレベルや状況を記入
"""

tasks = [
    { 'タスク名': 'タスク名', '説明': '詳細説明' },
]
```

実行：

```bash
python main.py
```

## 出力例

```
============================================================
タスク優先順位ランキング
============================================================

1位：Excel練習（総合スコア: 25.5）
理由：汎用性が高く習得コストが低いため今すぐ着手しやすい

2位：Git/GitHubの使い方（総合スコア: 25.5）
理由：汎用性が高く将来性も高い
```
